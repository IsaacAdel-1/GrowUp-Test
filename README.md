# اختبار النضج الشخصي — GrowUp

اختبار تفاعلي (React + Vite) يقيس مستوى النضج الوجداني عبر **٦ مبادئ / ٤٠ سؤالاً**، ويعرض نتيجة محلَّلة بالشارتات + دليل نمو عملي لكل مبدأ. تصميم عربي RTL ومتجاوب بالكامل.

**مقياس الإجابة:** ٤ = صحيح دائماً · ٣ = صحيح غالباً · ٢ = صحيح أحياناً · ١ = ليس صحيحاً مطلقاً.

**مستويات النضج:** رضيع < طفل < مراهق < راشد (وجدانياً).

## التشغيل

```bash
npm install
npm run dev      # وضع التطوير
npm run build    # نسخة الإنتاج داخل مجلد dist
npm run preview  # معاينة نسخة الإنتاج
```

## الصفحات والتنقّل

راوتر بسيط بالـ hash (بدون مكتبات) — كل صفحة لها رابط قابل للمشاركة وزر الرجوع يعمل:

| المسار | الصفحة |
|--------|--------|
| `#/quiz` | الاختبار (٦ مبادئ، الإجابات تُحفظ داخل الجلسة) |
| `#/result` | النتيجة: **التركيز على نتيجة كل مبدأ** + شارتا مقارنة (Radar + Bar مرتّب) + الأقوى/الأضعف |
| `#/report` | تقرير كامل قابل للطباعة/**تصدير PDF** (زر «حفظ / طباعة PDF» → اختر «حفظ كـ PDF») |
| `#/stages` | مراحل النمو الأربع مع شرح كل مرحلة |
| `#/guide` أو `#/guide/<principleId>` | **خطوات النمو العملية** لكل مبدأ + ممارسة أسبوعية |

الشريط العلوي (Navbar) فيه التابّات، وتاب «النتيجة» يُفتَح بعد إكمال الاختبار.

## أين أعدّل المحتوى؟ (كله في `src/data/`)

### 📄 `quizData.js` — الأسئلة والمبادئ
كل مبدأ:
```js
{
  id: 'principle-2',
  label: 'المبدأ الثاني',
  title: 'اكسر سلطان الماضي على حياتك',
  short: 'كسر الماضي',                 // الاسم المختصر في الشارتات والبطاقات
  icon: 'history',                     // eye|history|leaf|shield|droplet|heart|users|bulb|sparkle|briefcase
  color: '#f0913c',                    // لون المبدأ (يلوّن الشارت + بطاقته في النتيجة/الدليل/التقرير)
  thresholds: { adult: 20, adolescent: 15, child: 10 }, // حدود الدرجات (مفتاح التصحيح الرسمي)
  questions: [ '...', '...' ],
}
```
`scale` و`meta` هنا أيضاً. عدد الأسئلة يُحسب تلقائياً.

### 📄 `levels.js` — المستويات الأربعة ونصوصها التفسيرية
اسم كل مستوى، اللون (`tone`)، الوصف الكامل، والصفات. (من مفتاح الاختبار الرسمي.)

### 📄 `growthGuide.js` — دليل النمو لكل مبدأ
لكل مبدأ: `summary` (التعريف)، `whyItMatters`، `signs` (علامات الضعف)، `steps` (خطوات النمو)، `practice` (ممارسة أسبوعية). **هذا المحتوى التحليلي حرٌّ للتعديل بالكامل.**

## أين أعدّل حساب النتيجة؟

### 📄 `src/utils/scoring.js`
- `levelIdForScore(sum, thresholds)` — تحويل مجموع المبدأ إلى مستوى (رضيع/طفل/مراهق/راشد).
- `computeResults(...)` — يحسب نتيجة كل مبدأ + النسبة والمستوى العام + أضعف/أقوى مبدأ.

> حدود الدرجات نفسها موجودة داخل كل مبدأ في `quizData.js` (`thresholds`).

## الألوان والمظهر

- كل متغيّرات التصميم (الألوان، الحواف، الظلال) في: 📄 `src/styles/tokens.css`
- ألوان الشارتات (hex) في: 📄 `src/utils/palette.js` — **يجب أن تبقى متطابقة** مع نفس القيم في `tokens.css` (لأن Recharts لا يقرأ `var(--...)`).
- الـ Palette الحالية: أزرق حيوي (العلامة + راشد) · نيلي (مراهق) · كهرماني (طفل) · مرجاني (رضيع) — أربعة ألوان مميّزة. لتغييرها كلها، عدّل الملفين معاً.
- **شخصيات المستويات** (رضيع/طفل/مراهق/راشد) مرسومة SVG في 📄 `src/components/LevelCharacters.jsx` (تتلوّن بلون كل مستوى، وتظهر في سُلّم المراحل وكروت صفحة «مراحل النمو»).
- **نظامان للألوان:** (١) ألوان المستويات الأربعة (في `tokens.css`/`palette.js`) تُستخدم للمستويات والشخصيات وصفحة المراحل. (٢) لون لكل مبدأ (`color` في `quizData.js`) يُلوّن شارت الأعمدة وبطاقات المبادئ في النتيجة والدليل والتقرير — عشان تبان متنوّعة الألوان.

## المكوّنات (كلها قابلة لإعادة الاستخدام + CSS Modules معزول لكل مكوّن)

```
src/
├─ App.jsx                 الراوتر + الحالة (الإجابات، التنقّل)
├─ hooks/useHashRoute.js   راوتر الـ hash البسيط
├─ data/                   quizData · levels · growthGuide
├─ utils/scoring.js        حساب النتائج والمستويات
└─ components/
   ├─ Navbar.jsx           الشريط العلوي + التابّات
   ├─ QuizView.jsx         واجهة الاختبار (ترويسة + تقدّم + أسئلة + تنقّل)
   ├─ QuizHeader / ProgressBar / SectionCard / QuestionCard / RatingScale / QuizNav / QuizFooter
   ├─ ResultsView.jsx      صفحة النتيجة (تركيز على كل مبدأ)
   ├─ ReportView.jsx       التقرير الكامل القابل للطباعة/PDF
   ├─ StagesView.jsx       صفحة مراحل النمو
   ├─ GuideView.jsx        خطوات النمو العملية لكل مبدأ
   ├─ MaturityRadar.jsx    شارت الرادار (Recharts)
   ├─ MaturityBar.jsx      شارت أعمدة أفقية مرتّبة (مقارنة)
   ├─ MaturityLadder.jsx   سُلّم النضج (رضيع → راشد) — في صفحة المراحل
   ├─ ChartTooltip.jsx     تولتيب الشارتات
   └─ icons.jsx            مجموعة الأيقونات (SVG)
```

**تصدير PDF:** يعتمد على طباعة المتصفح (`window.print()` + أنماط `@media print`) — يعطي نصاً عربياً وشارتات حادّة وقابلة للتحديد. لا يحتاج أي مكتبة إضافية.

## ملاحظات

- الإجابات تُحفظ في `sessionStorage` فقط (تبقى عند تحديث الصفحة، وتُمسح عند إغلاق التبويب) — لا تُرسل لأي جهة.
- التصميم متجاوب من موبايل صغير (٣٢٠px) حتى الشاشات الكبيرة.
- الشارتات مبنية بـ **Recharts** (سلسلة واحدة، بلون العلامة، مع تولتيب عند المرور).
- الخط: **Tajawal** من Google Fonts (يمكن استبداله من `index.html`).
